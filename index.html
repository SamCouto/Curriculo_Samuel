Portfólio

Criei um projeto simples utilizando a linguagem Python, o qual pode ser encontrado no meu GitHub pessoal:

  • Vending-Machine
    •
Este projeto se trata de uma máquina automática de vendas, semelhante às máquinas de refrigerante que vemos em locais públicos.
Ao iniciarmos o programa, ele abre uma tela no terminal, exibindo a interface de vendas.
Lá o usuário pode escolher quais dos itens ele deseja comprar e, selecionando o mesmo, uma seção de pagamento é aberta, para que o mesmo possa pagar pelo produto. Há tambem um painel secreto para o administrador onde ele pode ver o lucro, repor e retirar itens do estoque.
